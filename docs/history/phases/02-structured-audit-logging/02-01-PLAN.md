| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-P01-01 | Information Disclosure | PayloadRedactor | mitigate | Default-deny allowlist (D-08); only ALLOWED_FIELDS frozenset fields survive redaction. No "redact by pattern" approach. |
| T-02-P01-02 | Tampering | AuditSink.emit | mitigate | Fire-and-forget (D-06) — sink failure never affects pipeline outcome. Errors logged at ERROR level only. |
| T-02-P01-03 | Spoofing | pipeline_id in events | mitigate | pipeline_id validated at pipeline construction per existing validation (^[a-zA-Z0-9_-]{1,128}$). Events carry the validated ID, not user input. |
| T-02-P01-SC | Tampering | pip install (stdlib only) | accept | No external packages introduced this phase. All stdlib. |

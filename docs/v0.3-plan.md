# Relay v0.3 — Implementation Plan
> Theme: Relay should work with whatever the framework builder is already using. No lock-in.
>
> This document translates the v0.3 section of the Design Document into a concrete, commit-by-commit implementation plan that follows all Relay Coding Rules.

---

## Prerequisite

All 🔴 CRITICAL fixes from `relay-bug-and-fix-plan.md` (FIX-001 through FIX-004) must be merged and `mypy --strict` must pass before any v0.3 work begins. v0.3 builds on the type system — errors inherited from v0.2 will multiply.

## What v0.3 Delivers

v0.3 implements **Layer 3 — Agent Runner** from the design document: a universal adapter layer that lets framework builders plug any LLM provider into Relay without changing any Relay internals.

### New module: `relay.runners`

| Class / Protocol | Purpose |
|---|---|
| `AgentRunner` (Protocol) | The single contract every adapter implements |
| `AgentOutput` (dataclass) | Normalised output from any adapter |
| `ContextSlice` (dataclass) | What the runner receives — a slice, not the full context |
| `RawSDKAdapter` | Wraps any callable `(messages) -> str` |
| `LangChainAdapter` | Wraps any LangChain `Runnable` |
| `CrewAIAdapter` | Wraps a CrewAI `Agent` |
| `AutoGenAdapter` | Wraps an AutoGen `AssistantAgent` (single-turn) |
| `LocalModelAdapter` | Targets any OpenAI-compatible REST endpoint (Ollama, vLLM) |
| `AdapterRegistry` | Named adapter registration and lookup |

### Modified: `relay.core_pipeline`

`execute_step_with_manifest` gains an optional `runner` parameter. When provided, the pipeline calls the runner instead of expecting the caller to have already called the agent externally.

## Data Model

### `ContextSlice`

The runner receives a slice, never the full context. This enforces the read boundary defined in `AgentManifest`.

```python
# src/relay/runners/protocol.py

from dataclasses import dataclass
from typing import Any


@dataclass(frozen=True)
class ContextSlice:
    """A read-only, bounded view of context delivered to a single agent.

    Owns: the data a specific agent is permitted to read for one step.
    Does NOT: contain history, the full envelope, or sections outside the manifest.
    """

    pipeline_id: str
    step: int
    agent_id: str
    sections: dict[str, Any]      # key → content, already filtered by SlicePacker
    token_count: int               # estimated cost of this slice
    manifest_hash: str             # hash of the manifest that produced this slice
```

### `AgentOutput`

All adapters normalise their response to this schema. The handoff validator runs on `AgentOutput`, not on raw adapter responses.

```python
@dataclass(frozen=True)
class AgentOutput:
    """Normalised output from any agent adapter.

    Owns: the agent's response in a uniform, Relay-compatible format.
    Does NOT: contain raw provider responses, streaming data, or tool internals.
    """

    text: str
    structured: dict[str, Any]    # empty dict if adapter produced plain text
    tool_calls: list[dict[str, Any]]  # empty list if no tool use
    token_count: int               # tokens consumed by the agent call
    latency_ms: int                # wall-clock time of the call in milliseconds
    adapter: str                   # name of the adapter that produced this output
```

### `AgentRunner` Protocol

```python
from typing import Protocol, runtime_checkable


@runtime_checkable
class AgentRunner(Protocol):
    """Protocol for all agent adapters.

    Every adapter implements exactly this interface.
    The runner receives a slice (not the full context) and the agent's manifest.
    It returns a structured AgentOutput.
    Everything else — prompt construction, retries, streaming — is the adapter's job.
    """

    async def run(
        self,
        slice: ContextSlice,
        manifest: "AgentManifest",
    ) -> AgentOutput:
        """Execute one agent turn and return normalised output."""
        ...
```

**Why async:** All adapters are async from the start for forward compatibility with v0.4 (parallel execution). Synchronous callers wrap with `asyncio.run()`. Synchronous adapters are wrapped with `asyncio.to_thread()` inside an async shim.

## Adapter Implementations

### `RawSDKAdapter` — `src/relay/runners/raw_sdk.py`

Wraps any callable that takes a list of messages and returns a string. No framework required. This is the adapter for teams using Anthropic, OpenAI, or any other SDK directly.

```python
"""RawSDKAdapter — wraps a plain callable as an AgentRunner.

Owns: converting ContextSlice to a message list and calling the provided callable.
Does NOT: manage retries, streaming, or tool parsing.
"""

import asyncio
import time
from dataclasses import dataclass
from typing import Callable, Awaitable

from relay.runners.protocol import AgentOutput, AgentRunner, ContextSlice
from relay.slicer.manifest import AgentManifest


MessageList = list[dict[str, str]]
SyncCallable  = Callable[[MessageList], str]
AsyncCallable = Callable[[MessageList], Awaitable[str]]


@dataclass
class RawSDKAdapter:
    """Adapter for any callable that accepts messages and returns a string.

    Owns: message formatting and callable invocation.
    Does NOT: manage LLM sessions, prompts, or retries.

    Supports both sync and async callables. Sync callables are run via
    asyncio.to_thread to keep the runner interface uniformly async.
    """

    callable: SyncCallable | AsyncCallable
    adapter_name: str = "raw_sdk"

    def _build_messages(self, slice: ContextSlice) -> MessageList:
        """Convert ContextSlice to OpenAI-style message list."""
        import json
        context_text = json.dumps(slice.sections, indent=2)
        return [{"role": "user", "content": context_text}]

    async def run(self, slice: ContextSlice, manifest: AgentManifest) -> AgentOutput:
        """Execute the callable and normalise the response."""
        messages = self._build_messages(slice)
        start = time.monotonic()

        result = self.callable(messages)
        if asyncio.iscoroutine(result):
            text = await result
        else:
            text = await asyncio.to_thread(lambda: result)  # type: ignore[arg-type]

        latency_ms = int((time.monotonic() - start) * 1000)
        # Token count: use slice.token_count as a lower bound; output adds ~output_chars // 4
        output_tokens = len(text) // 4
        return AgentOutput(
            text=text,
            structured={},
            tool_calls=[],
            token_count=slice.token_count + output_tokens,
            latency_ms=latency_ms,
            adapter=self.adapter_name,
        )
```

### `LangChainAdapter` — `src/relay/runners/langchain.py`

```python
"""LangChainAdapter — wraps a LangChain Runnable as an AgentRunner.

Owns: converting ContextSlice to LangChain input format and invoking the Runnable.
Does NOT: configure the Runnable, manage LangChain memory, or handle streaming.
"""

import time
from dataclasses import dataclass
from typing import Any

from relay.runners.protocol import AgentOutput, ContextSlice
from relay.slicer.manifest import AgentManifest


@dataclass
class LangChainAdapter:
    """Wraps any LangChain Runnable (chain, agent, or LCEL pipeline).

    Owns: input conversion and output normalisation.
    Does NOT: manage LangChain memory — Relay owns memory via SnapshotStore.

    LangChain's internal memory is deliberately not used when running under Relay.
    Pass a stateless Runnable. History is Relay's job.
    """

    runnable: Any   # langchain_core.runnables.Runnable — not imported at module level
    adapter_name: str = "langchain"

    def _build_input(self, slice: ContextSlice) -> dict[str, Any]:
        """Convert ContextSlice to LangChain input dict."""
        import json
        return {
            "input": json.dumps(slice.sections, indent=2),
            "agent_id": slice.agent_id,
            "step": slice.step,
        }

    async def run(self, slice: ContextSlice, manifest: AgentManifest) -> AgentOutput:
        """Invoke the LangChain Runnable and normalise output."""
        import asyncio, json

        lc_input = self._build_input(slice)
        start = time.monotonic()

        # LangChain Runnables expose ainvoke for async
        if hasattr(self.runnable, "ainvoke"):
            response = await self.runnable.ainvoke(lc_input)
        else:
            import asyncio
            response = await asyncio.to_thread(self.runnable.invoke, lc_input)

        latency_ms = int((time.monotonic() - start) * 1000)

        # Normalise: LangChain may return str, dict, AIMessage, etc.
        if isinstance(response, str):
            text = response
            structured: dict[str, Any] = {}
        elif isinstance(response, dict):
            text = response.get("output", json.dumps(response))
            structured = {k: v for k, v in response.items() if k != "output"}
        else:
            # AIMessage or other LangChain object — try .content attribute
            text = getattr(response, "content", str(response))
            structured = {}

        output_tokens = len(text) // 4
        return AgentOutput(
            text=text,
            structured=structured,
            tool_calls=[],          # tool call extraction is adapter-specific; extend if needed
            token_count=slice.token_count + output_tokens,
            latency_ms=latency_ms,
            adapter=self.adapter_name,
        )
```

### `CrewAIAdapter` — `src/relay/runners/crewai.py`

```python
"""CrewAIAdapter — wraps a CrewAI Agent as an AgentRunner.

Owns: invoking the CrewAI agent for a single turn and normalising output.
Does NOT: manage CrewAI crew lifecycle, inter-agent communication, or CrewAI memory.

IMPORTANT: CrewAI's internal memory is disabled by default when running under Relay.
Relay owns memory via SnapshotStore. Enabling CrewAI memory creates an untracked
second state store — a direct violation of Relay's design principles.
"""

import asyncio
import json
import time
from dataclasses import dataclass, field
from typing import Any

from relay.runners.protocol import AgentOutput, ContextSlice
from relay.slicer.manifest import AgentManifest


@dataclass
class CrewAIAdapter:
    """Wraps a CrewAI Agent for single-turn execution under Relay.

    Owns: task construction, single-turn invocation, output normalisation.
    Does NOT: manage CrewAI crews, enable CrewAI memory, or handle multi-turn loops.
    """

    agent: Any    # crewai.Agent — not imported at module level (optional dep)
    adapter_name: str = "crewai"
    _memory_warning_issued: bool = field(default=False, init=False, repr=False)

    def __post_init__(self) -> None:
        """Warn if CrewAI memory is enabled — it creates an untracked state store."""
        if getattr(self.agent, "memory", False):
            import warnings
            warnings.warn(
                "CrewAI agent has memory enabled. Relay owns memory via SnapshotStore. "
                "Disable CrewAI memory to prevent untracked state divergence.",
                stacklevel=2,
            )

    def _build_task_description(self, slice: ContextSlice) -> str:
        return f"Context for step {slice.step}:\n{json.dumps(slice.sections, indent=2)}"

    async def run(self, slice: ContextSlice, manifest: AgentManifest) -> AgentOutput:
        """Execute one CrewAI agent turn."""
        try:
            from crewai import Task
        except ImportError:
            raise ImportError(
                "crewai is required for CrewAIAdapter. "
                "Install with: pip install relay-middleware[crewai]"
            )

        task = Task(description=self._build_task_description(slice), agent=self.agent)
        start = time.monotonic()

        # CrewAI execute_sync runs synchronously — wrap in to_thread
        response = await asyncio.to_thread(task.execute_sync)
        latency_ms = int((time.monotonic() - start) * 1000)

        text = str(response) if not isinstance(response, str) else response
        output_tokens = len(text) // 4
        return AgentOutput(
            text=text,
            structured={},
            tool_calls=[],
            token_count=slice.token_count + output_tokens,
            latency_ms=latency_ms,
            adapter=self.adapter_name,
        )
```

### `AutoGenAdapter` — `src/relay/runners/autogen.py`

```python
"""AutoGenAdapter — wraps an AutoGen agent for single-turn execution under Relay.

Owns: constructing a single-turn AutoGen conversation and extracting the delta.
Does NOT: manage multi-turn AutoGen loops, enable AutoGen history, or configure groupchats.

AutoGen's conversation loop is single-stepped: Relay calls one turn at a time
and captures only the delta — the new message produced in this turn — not the
full AutoGen history.
"""

import asyncio
import json
import time
from dataclasses import dataclass
from typing import Any

from relay.runners.protocol import AgentOutput, ContextSlice
from relay.slicer.manifest import AgentManifest


@dataclass
class AutoGenAdapter:
    """Wraps an AutoGen AssistantAgent or UserProxyAgent for single-turn use.

    Owns: single-turn invocation and delta extraction.
    Does NOT: maintain AutoGen conversation history between Relay steps.
    """

    agent: Any          # autogen.AssistantAgent — not imported at module level
    adapter_name: str = "autogen"

    def _make_user_proxy(self) -> Any:
        """Create a minimal UserProxyAgent for single-turn invocation."""
        try:
            import autogen
        except ImportError:
            raise ImportError(
                "pyautogen is required for AutoGenAdapter. "
                "Install with: pip install relay-middleware[autogen]"
            )
        return autogen.UserProxyAgent(
            name="relay_proxy",
            human_input_mode="NEVER",
            max_consecutive_auto_reply=1,
            code_execution_config=False,
        )

    async def run(self, slice: ContextSlice, manifest: AgentManifest) -> AgentOutput:
        """Execute one AutoGen agent turn, capturing only the new delta."""
        user_proxy = self._make_user_proxy()
        message = json.dumps(slice.sections, indent=2)
        start = time.monotonic()

        # initiate_chat is synchronous — wrap in to_thread
        await asyncio.to_thread(
            user_proxy.initiate_chat,
            self.agent,
            message=message,
            max_turns=1,
        )

        latency_ms = int((time.monotonic() - start) * 1000)

        # Extract only the last message (the delta from this turn)
        chat_history = user_proxy.chat_messages.get(self.agent, [])
        if chat_history:
            last_msg = chat_history[-1]
            text = last_msg.get("content", "") if isinstance(last_msg, dict) else str(last_msg)
        else:
            text = ""

        output_tokens = len(text) // 4
        return AgentOutput(
            text=text,
            structured={},
            tool_calls=[],
            token_count=slice.token_count + output_tokens,
            latency_ms=latency_ms,
            adapter=self.adapter_name,
        )
```

### `LocalModelAdapter` — `src/relay/runners/local_model.py`

```python
"""LocalModelAdapter — targets any OpenAI-compatible REST endpoint.

Owns: HTTP call construction and response normalisation for local model servers.
Does NOT: manage model loading, GPU resources, or server lifecycle.

Compatible with Ollama and vLLM. No streaming support in v0.3.
"""

import json
import time
from dataclasses import dataclass, field
from typing import Any

from relay.runners.protocol import AgentOutput, ContextSlice
from relay.slicer.manifest import AgentManifest
from relay.types import Failure, Result, Success


@dataclass
class LocalModelAdapter:
    """Adapter for local model servers exposing an OpenAI-compatible REST API.

    Owns: HTTP request construction and response normalisation.
    Does NOT: manage model lifecycle, GPU resources, or server processes.

    Example:
        adapter = LocalModelAdapter(
            base_url="http://localhost:11434",
            model="llama3",
        )
    """

    base_url: str
    model: str
    adapter_name: str = "local_model"
    timeout_seconds: float = 60.0

    def __post_init__(self) -> None:
        # Strip trailing slash to avoid double-slash in URL construction
        object.__setattr__(self, "base_url", self.base_url.rstrip("/"))

    def _build_payload(self, slice: ContextSlice) -> dict[str, Any]:
        context_text = json.dumps(slice.sections, indent=2)
        return {
            "model": self.model,
            "messages": [{"role": "user", "content": context_text}],
            "stream": False,
        }

    async def run(self, slice: ContextSlice, manifest: AgentManifest) -> AgentOutput:
        """POST to the local model endpoint and normalise the response."""
        try:
            import httpx
        except ImportError:
            raise ImportError(
                "httpx is required for LocalModelAdapter. "
                "Install with: pip install relay-middleware[local]"
            )

        payload = self._build_payload(slice)
        url = f"{self.base_url}/v1/chat/completions"
        start = time.monotonic()

        async with httpx.AsyncClient(timeout=self.timeout_seconds) as client:
            response = await client.post(url, json=payload)
            response.raise_for_status()
            data = response.json()

        latency_ms = int((time.monotonic() - start) * 1000)

        # OpenAI-compatible response schema
        choices = data.get("choices", [])
        text = choices[0]["message"]["content"] if choices else ""
        usage = data.get("usage", {})
        token_count = usage.get("total_tokens", slice.token_count + len(text) // 4)

        return AgentOutput(
            text=text,
            structured={},
            tool_calls=[],
            token_count=token_count,
            latency_ms=latency_ms,
            adapter=self.adapter_name,
        )
```

### `AdapterRegistry` — `src/relay/runners/registry.py`

```python
"""AdapterRegistry — named adapter registration and lookup.

Owns: mapping adapter names to AgentRunner instances.
Does NOT: construct adapters, manage LLM sessions, or validate adapter behaviour.
"""

from dataclasses import dataclass, field
from typing import Any

from relay.runners.protocol import AgentRunner
from relay.types import Failure, Result, Success


@dataclass
class AdapterRegistry:
    """Registry of named AgentRunner instances.

    Framework builders register adapters at pipeline initialisation.
    Steps look up adapters by name at execution time.

    Example:
        registry = AdapterRegistry()
        registry.register("summariser", LangChainAdapter(my_chain))
        registry.register("extractor", RawSDKAdapter(my_fn))
    """

    _adapters: dict[str, AgentRunner] = field(default_factory=dict, init=False, repr=False)

    def register(self, name: str, adapter: AgentRunner) -> None:
        """Register an adapter under a given name.

        Args:
            name: Identifier used to look up this adapter in pipeline steps.
            adapter: Any object satisfying the AgentRunner Protocol.

        Raises:
            ValueError: If name is empty or adapter does not satisfy AgentRunner.
        """
        if not name:
            raise ValueError("Adapter name cannot be empty")
        if not isinstance(adapter, AgentRunner):
            raise ValueError(
                f"Object of type {type(adapter).__name__} does not satisfy AgentRunner protocol. "
                "Implement an async run(slice, manifest) -> AgentOutput method."
            )
        self._adapters[name] = adapter

    def get(self, name: str) -> Result[AgentRunner]:
        """Look up an adapter by name.

        Returns:
            Success with the adapter, or Failure if not registered.
        """
        adapter = self._adapters.get(name)
        if adapter is None:
            return Failure(
                reason=f"No adapter registered under name '{name}'",
                code="ADAPTER_NOT_FOUND",
            )
        return Success(adapter)

    def list_names(self) -> list[str]:
        """Return sorted list of registered adapter names."""
        return sorted(self._adapters.keys())
```

## Pipeline Integration

### Changes to `core_pipeline.py`

Add an optional `runner` parameter to `execute_step_with_manifest`. When provided, the pipeline calls the runner instead of expecting pre-produced agent output:

```python
# In CoreRelayPipeline:
registry: Optional[AdapterRegistry] = None   # new field

async def execute_step_with_runner(
    self,
    adapter_name: str,
    manifest: AgentManifest,
) -> Result[ContextEnvelope]:
    """Execute a pipeline step by running the named adapter.

    The pipeline:
      1. Slices current context per manifest
      2. Checks budget
      3. Calls the registered adapter
      4. Validates manifest boundaries on the output
      5. Creates and commits the next envelope

    Args:
        adapter_name: Name of the adapter in the registry to invoke.
        manifest: Agent manifest defining read/write permissions.

    Returns:
        Success with the new envelope, or Failure / RollbackSuccess.
    """
    if self.registry is None:
        return Failure(reason="No AdapterRegistry configured", code="NO_REGISTRY")

    adapter_result = self.registry.get(adapter_name)
    if isinstance(adapter_result, Failure):
        return adapter_result
    adapter = adapter_result.value

    current = self.get_current_envelope()
    slice_ = self._build_context_slice(current, manifest)

    budget_result = self._check_budget(manifest, current)
    if isinstance(budget_result, Failure):
        return budget_result

    agent_output = await adapter.run(slice_, manifest)

    # Convert AgentOutput to the dict that execute_step_with_manifest expects
    payload = {"text": agent_output.text, **agent_output.structured}
    return self.execute_step_with_manifest(payload, manifest=manifest)
```

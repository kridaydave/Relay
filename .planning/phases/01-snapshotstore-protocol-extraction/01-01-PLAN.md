## Description

Extract the public method signatures of `SnapshotStore` into a standalone `@runtime_checkable` Protocol, rename the file-based class to `LocalFileSnapshotStore`, and update all consumers to import the Protocol from its new home. Tests are updated to reference `LocalFileSnapshotStore` for concrete instantiation and `SnapshotStore` for type checks. The Protocol extends `Closeable` (from `relay.types`).


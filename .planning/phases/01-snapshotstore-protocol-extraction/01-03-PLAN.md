## Description

Add an optional `snapshot_store: SnapshotStore | None = None` field parameter to `CoreRelayPipeline`, following the same injection pattern as `token_counter`, `slice_packer`, and `registry`. When `None`, the existing default `LocalFileSnapshotStore(storage_path)` is constructed. When provided, the custom store is used directly. Also update `create()` factory and `close()` to handle the injected store.
